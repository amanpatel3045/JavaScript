# JavaScript Assignment Operators

## Assignment operators assign values to JavaScript variables.

The Addition Assignment Operator (+=) adds a value to a variable.

```
let x = 10;
x += 5;
```

```
Operator	Example	Same As
=	x = y	x = y
+=	x += y	x = x + y
-=	x -= y	x = x - y
*=	x *= y	x = x * y
/=	x /= y	x = x / y
%=	x %= y	x = x % y
**=	x **= y	x = x ** y
```
